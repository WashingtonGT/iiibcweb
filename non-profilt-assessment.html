How to use this assessment

Designed for Executive Directors, Program Directors, COOs, and Board Technology Committees. Each question is scored 0–3: 0 = not in place / no awareness · 1 = awareness only · 2 = in progress / partial · 3 = fully in place and practiced. There are no trick questions — this is a diagnostic, not a grade.
