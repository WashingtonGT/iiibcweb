## Key Navigation Paths

### For Technology-Driven Business Leaders
1. Land on **index.html** → Hero
2. Explore **#segments** (Beachhead 1 details)
3. Review **#services** (Tiers 1-2 most relevant)
4. Read **#why-iiibc** comparison
5. Book **#diagnostic** or review **go-to-market-strategy.html**

### For Non-Profit/Advocacy Leaders
1. Land on **index.html** → Urgency ticker
2. Review **#segments** (Beachhead 2 — DC focus)
3. Understand risks in **#segments**
4. Explore **#services** (Tiers 1 & 3 most relevant)
5. Access **non-profit-training.html** or **non-profilt-assessment.html**
6. Book **#diagnostic**

### For Investors / Partners
1. Start at **go-to-market-strategy.html**
2. Review **#services** on index
3. Understand **#why-iiibc** positioning
4. Review **#partnerships** for channel program
5. Check **#founding-client** program details

### For CAIOs / AI Leaders
1. Browse **#digest** preview
2. Access latest **digest-2026-06-04.html**
3. Review **#services** (Tiers 2-4)
4. Check **IIIBC-Executive-Briefing.html**
5. Book **#diagnostic** for custom assessment
